# Values - ценности и видение

## Видение продукта

- VIBE-SERCH = стандарт индустрии: главный утренний дайджест для AI-сообщества, как Hacker News для AI
- Информация должна быть проверена, а не просто агрегирована
- Каждый факт проходит мультиагентную верификацию - это то, что отличает от простых агрегаторов
- Пользователь доверяет дайджесту как первоисточнику: если VIBE-SERCH сказал - значит так и есть
- Продукт из личной боли: сам хочу такой инструмент, существующие не решают мою проблему
- Telegram как привычный канал - не заставляем менять поведение
- Один качественный дайджест утром > бесконечный поток непроверенных новостей

## Ценности в работе

- Скорость итераций: быстрее выпустить, получить фидбэк, улучшить - цикл короче = результат ближе
- Честность с пользователем: не преувеличивать, не скрывать ограничения, быть прозрачным
- Простота решений: простое побеждает элегантное, минимум сложности - максимум понятности
- Автоматизация рутины: если делаешь дважды - автоматизируй
- Каждая цифра - со ссылкой на источник, каждое утверждение - с примером
- Экономность: укладываемся в $50/мес на инфра, оптимизируем расходы на API

## Антиценности (чего избегаем)

- Overengineering - не строим архитектуру на 1M пользователей когда нет 100
- Хайп и кликбейт - даже в маркетинге своего продукта
- Перфекционизм - лучше запустить сырое чем не запустить идеальное
- Реклама и спонсорский мусор в дайджесте
