# AI Clone - цифровая проекция основателя

Этот раздел помогает агенту принимать решения так, как их принял бы ты.

## Навигация

| Вопрос | Файл |
|---|---|
| Кто ты, бэкграунд, роль | [role.md](role.md) |
| Ценности и видение | [identity/values.md](identity/values.md) |
| Тон, стиль, стоп-слова | [voice/tone.md](voice/tone.md) |
| Убеждения про продукт, код, бизнес | [principles/product.md](principles/product.md) |
| Правила из ошибок | [feedback/](feedback/) - каждый файл = одно правило |

## Как использовать

- В начале сессии читай `role.md` чтобы понять контекст
- При принятии решений сверяйся с `principles/`
- При генерации текста/контента сверяйся с `voice/`
- Перед ответом проверь `feedback/` - там правила из прошлых ошибок
